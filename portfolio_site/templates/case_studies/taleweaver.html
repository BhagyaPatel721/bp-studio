TaleWeaver

AI-Powered Storytelling Platform

Django LLM (LLaMA-3) RAG FAISS

Project Overview

TaleWeaver is a full-stack AI storytelling platform that generates high-quality stories using retrieval-augmented generation and large language models, wrapped in a modern, ChatGPT-style interface.

The Problem

  • AI stories felt generic and repetitive
  • No contextual memory across prompts
  • Poor UI/UX in existing storytelling platforms

Design & Development Approach

I designed TaleWeaver with a focus on clean UI, smooth interactions, and scalable backend architecture. The system combines curated datasets, vector embeddings, and LLaMA-3 to deliver context-aware storytelling.

Technology Stack

Python
Groq API
LLaMA-3 70B

HTML · CSS · JavaScript

Key Features

Chat-Based Story Generation
Genre-Specific Prompting
Story Regeneration
PDF & Word Downloads
User Authentication
Persistent Chat History

Outcome & Impact

TaleWeaver stands as a production-ready AI application demonstrating advanced backend engineering, modern UI/UX design, and scalable AI integration — built as a 500-mark final year project.
